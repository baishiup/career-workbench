# 任务 3：job_parse 工作流接入

## 目标

admin 粘贴 JD 文本，AI 解析成结构化字段预填导入表单，人工确认后保存。

## 前置条件

- 任务 2（导入表单已存在）。

## 边界

- 只做文本解析；链接自动抓取和截图 OCR 不做。
- 解析失败必须能降级为纯手动填写。

## 输入

- 任务 2 的导入表单。
- `dify/` 现有简历解析工作流，作为编排参考。

## 实现

- Dify `job_parse` workflow：JD 文本 → JSON schema 输出，字段对齐 `job_descriptions` 表。
- Edge Function 调用 + 鉴权（仅 admin）。
- 表单增加"粘贴 JD 解析"入口，解析结果预填、人工校正后提交。
- `importStatus` 流转：已解析 / 待人工确认 / 解析失败可重试。

## 验收

- 粘贴文本能预填表单。
- Dify 失败时显示错误并可重试或转手动填写。
- 三种导入状态都能产生并正确展示。
